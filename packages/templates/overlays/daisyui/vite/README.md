# daisyui/vite overlay

Minimal DaisyUI overlay for Vite frontends.

## Dependency additions

- devDependencies:
  - `tailwindcss`
  - `postcss`
  - `autoprefixer`
  - `daisyui`

## Files injected

- `tailwind.config.ts`
- `postcss.config.cjs`
- `src/styles.overlay.css`
- `src/neobrutalist-theme.css`
- `src/neobrutalist-ui.css`

## Style parity

Theme classes and component rules match the canonical neobrutalist baseline.
